# AirHost&ShortStay (AH&SS)

## Production‑ready frontend for a real‑time vacation rental marketplace

### Features
- Uber‑style real‑time discovery map
- HomeToGo split layout (filters + listings + map)
- Incentive engine with scoring, badges, fraud detection
- Lipa Mdogo (installment payments) and negotiation
- Wishlist, comparison, and role‑based dashboards
- Progressive Web App (offline support)
- Fully modular vanilla JavaScript (ES6+)

### Tech stack
- HTML5, CSS3 (no frameworks)
- Leaflet for maps (OpenStreetMap)
- Vanilla JavaScript with modular architecture
- LocalStorage for client‑side persistence
- Service Worker for offline caching

### Deployment
Ready for GitHub Pages or any static hosting. Backend API required for full functionality.
